How did i get through it

  1. create/download a web-template Code of Conduct.
  2. you can use open-source service worker like sw-toolbox.js provided by google Slack, clone this js file to your project
  3. Now create your js file to connect to that sw-toolbox and paste following:
    'use strict'; importScripts('sw-toolbox.js'); toolbox.precache(["index.html","style/style.css"]); toolbox.router.get('/images/*', toolbox.cacheFirst); toolbox.router.get('/*', toolbox.networkFirst, { networkTimeoutSeconds: 5 });
  4. NOw create manifest file for your website with some default icon
  5. Now host your app and check the Performance Using Google lightHouse tool extension
